         O objetivo da programação de um robô seguidor de linha é manter os dois sensores sobre a linha. A lógica básica de nossa programação é: Caso os dois sensores estejam sobre a linha, ande para frente; Caso um sensor esteja sobre a linha e o outro não, devemos fazer uma curva para corrigir o percurso; Caso ambos os sensores não estejam sobre a linha, ele deve estar parado. Para tanto, são utilizados 2 sensores infravermelhos que detectam a presença ou a ausência da linha e, de acordo com a combinação dos mesmos, o carrinho irá para frente ou virar para um dos lados.
      É importante ressaltar que o driver para motores é essencial para o funcionamento do robô, já que se ligar os motores diretamente nos pinos digitais do Arduino, estes pinos irão queimar - pois a corrente elétrica que os motores precisam para funcionar é maior do que a máxima corrente que os pinos do Arduino conseguem fornecer. Um exemplo de driver de motor muito utilizado para tal projeto é a Ponte H.
      Em cima destas duas premissas, o código do robô será desenvolvido através da leitura constante dos sensores e enviando o comando correto para os motores através do driver da placa, como mostra a imagem ao lado.
